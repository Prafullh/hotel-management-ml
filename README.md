# Hotel Management ML Project

## Overview
This project uses Machine Learning to predict hotel booking cancellations before they occur.  
The system analyzes hotel booking data and identifies patterns that help hotels reduce revenue loss and improve booking management.

---

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Machine Learning prediction model
- Booking cancellation prediction
- Accuracy evaluation
- Visualization of important insights

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

---

## Dataset
The dataset contains hotel booking information such as:
- Lead time
- Arrival date
- Number of guests
- Deposit type
- Booking status
- Hotel type

---

## Machine Learning Models
This project  include:
- Logistic Regression
- Random Forest
- XGBoost


---

## Project Goal
The main goal is to help hotels:
- Predict booking cancellations
- Improve resource planning
- Reduce revenue loss
- Improve customer management

---

## How To Run
1. Clone the repository
2. Open the notebook in Google Colab or Jupyter Notebook
3. Install required libraries
4. Run all notebook cells
